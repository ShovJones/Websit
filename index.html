What are CSSBattles?

CSS Battles are code-golfing games meant to test the skills of developers across the world. Coders use tbeir CSS skills to

replicate targets with smallest possible code. There are multiple levels, increasing in difficulty as one goes along.

This site contains some of the explanations to the first few battles.
